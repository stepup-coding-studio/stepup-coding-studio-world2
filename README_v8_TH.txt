STEP UP CODING STUDIO WORLD 2 — iPad Fix v8

แก้ไขจากผลทดสอบจริง:
1) ปุ่ม Command Palette ทั้ง 5 ปุ่มมีความสูงเท่ากัน
2) ลูกศรเดินขึ้น/ลง/ซ้าย/ขวาใหญ่และเส้นหนาขึ้น
3) แก้ระบบเปิดเพลงบน Safari/iPad เพื่อลดปัญหาการสั่งเปิดแล้วถูกสั่งปิดซ้ำ
4) เพิ่มระดับเสียงเพลงพื้นหลังให้ได้ยินชัดขึ้น
5) คงตำแหน่งปุ่ม RUN / UNDO / ล้าง / คำใบ้ตามเวอร์ชัน v7

วิธีทดสอบเสียงบน iPad:
- เปิดหน้าเว็บแล้วแตะ START หรือแตะพื้นที่เกม 1 ครั้ง
- ตรวจว่าไอคอนโน้ตเพลงด้านบนทำงาน
- หากปิดเพลงไว้ ให้แตะไอคอนโน้ตเพลง 1 ครั้ง
