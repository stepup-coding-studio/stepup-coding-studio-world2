แก้ไขตามภาพทดสอบล่าสุด
1. Coding Area ทำเป็นพื้นที่โล่ง ไม่มีเส้นแบ่งภายใน
2. ปุ่ม RUN / UNDO / ล้าง / คำใบ้ ใหญ่ขึ้น
3. ตัวหนังสือใน Command Palette ใหญ่ขึ้น
4. ลูกศรคำสั่งเดินใหญ่ขึ้น
5. ปุ่มทำซ้ำและข้อความทำซ้ำใหญ่ขึ้น
6. คำแนะนำการวางทำซ้ำอยู่ล่างสุดใต้ปุ่มทำซ้ำ
7. คง Layout หน้าเดียวสำหรับ iPad ไม่มีการเลื่อนขึ้นลง
