วิธีทดสอบบน Windows
1. แตกไฟล์ ZIP
2. เปิดโฟลเดอร์ STEP_UP_CODING_STUDIO_WORLD2_FINAL_GITHUB_READY
3. ดับเบิลคลิก index.html
4. ทดสอบ START, การลาก, RUN, เสียง และการเปลี่ยน Episode

วิธีใส่ GitHub Desktop
1. เปิดโฟลเดอร์ Repository: stepup-coding-studio-world2-test
2. ลบเฉพาะไฟล์เกมทดลองเก่าใน Repository (อย่าลบโฟลเดอร์ .git)
3. คัดลอกไฟล์ทั้งหมดจากโฟลเดอร์ชุดนี้ไปวางใน Repository
4. กลับไป GitHub Desktop
5. ใส่ Summary: World 2 Final 20 Episodes
6. กด Commit to main
7. กด Publish repository หรือ Push origin

เปิด GitHub Pages
1. เข้า Repository บนเว็บไซต์ GitHub
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: main
5. Folder: /(root)
6. Save
7. รอประมาณ 1–5 นาที แล้วเปิดลิงก์ GitHub Pages บน iPad
